# dotfiles

## Overview

I was exhausted from repeated environmental creation.  
So I decided to upload dotfiles to github.  

There are descriptions about vim and zsh, tmux settings.

## Requirements

- Git
- Python 3.X
- Zsh
- Tmux
- Oh-My-Zsh
- Powerline

### Options

- neovim
- peco
- exa
- ripgrep
- ccat
- colordiff

## Installation

```shell
git clone https://github.com/mnrn/dotfiles
cd dotfiles
python links.py
```
